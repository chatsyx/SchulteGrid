基于 Python+PyQt6 开发的轻量级舒尔特方格注意力训练工具，桌面部署直接可用，支持多规格网格切换、精准计时与标准评级，无冗余依赖，适配Windows系统，开源供个人学习与自用。

## 项目版本
当前发布版本：v1.0（基础版，仅源代码发布）
后续迭代计划：v1.0.1（新增exe可执行文件，无Python环境也能运行）、v1.1（新增进阶训练功能）

## 核心功能
1.  多规格网格：支持3×3（入门）、4×4（基础）、5×5（标准）、6×6（进阶）、7×7（高阶），适配不同人群注意力训练需求
2.  精准计时：毫秒级计时，结果保留2位小数，真实记录训练耗时，无误差
3.  顺序校验：点击数字错序自动弹窗提醒，正确点击后按钮失效，避免重复操作，保障训练有效性
4.  标准评级：5×5规格匹配行业通用训练标准，自动判定训练等级（优秀/良好/一般/需加强），直观反馈训练效果
5.  极简界面：无广告无冗余元素，贴合注意力训练场景，界面整洁不干扰训练状态，操作简单易上手

## 训练标准（5×5规格 官方通用参考）
- 优秀：25~30秒
- 良好：30~40秒
- 一般：40~50秒
- 需加强：50秒以上

## 环境部署（Windows桌面版，步骤极简，新手友好）
### 前置要求
本地需安装 Python 3.8+ 版本（安装时务必勾选「Add Python to PATH」，避免后续配置环境变量）

### 部署步骤
1.  克隆仓库（替换为你的GitHub仓库HTTPS地址）
```bash
git clone https://github.com/chatsyx/SchulteGrid.git
进入项目根目录（仓库默认部署在桌面，直接执行以下命令即可精准定位，无需手动找路径）
bash
运行
cd $env:USERPROFILE\Desktop\SchulteGrid
安装项目唯一依赖（仅需安装 PyQt6，网络正常即可快速完成）
bash
运行
pip install pyqt6
启动项目（命令简单，一键运行）
bash
运行
python main.py
使用教程（零基础也能会）
项目启动后，在顶部下拉框选择训练网格规格，默认是 3×3 入门规格
点击「开始训练」按钮，界面自动生成乱序数字网格，计时进入待启动状态
按 1→2→3→...→n² 的顺序依次点击数字，错序会弹出提示框提醒，正确点击后对应按钮变灰，无法重复点击
点击最后 1 个数字后，自动停止计时，弹窗显示本次训练总用时，5×5 规格同步显示对应训练等级
训练完成后，可点击「重置网格」清空当前界面重新训练，也可直接切换网格规格开展不同难度训练
常见问题排查（新手必看，规避 90% 踩坑）
问题 1：运行命令报错「找不到模块 PyQt6」
解决方案：网络波动导致依赖安装失败，重新执行安装命令即可
bash
运行
pip install pyqt6
问题 2：运行报错「TypeError: addLayout () got an unexpected keyword argument 'alignment'」
解决方案：该问题为 PyQt6 方法使用差异，v1.0 版本已完成修复，直接拉取仓库最新代码，无需额外修改文件
问题 3：运行命令报错「Python 不是内部或外部命令，也不是可运行的程序或批处理文件」
解决方案：Python 安装时未勾选「Add Python to PATH」，需重新安装 Python 并勾选该选项，或手动配置系统环境变量
问题 4：网格数字按钮显示不全、排版错乱
解决方案：打开 ui/main_ui.py 文件，将代码中 btn.setFixedSize(60, 60) 修改为 btn.setFixedSize(70, 70)，保存文件后重新运行项目即可
问题 5：点击「开始训练」无反应
解决方案：未完成网格规格切换确认，切换规格后直接点击开始即可，无需额外操作；若仍无反应，重启项目即可恢复
